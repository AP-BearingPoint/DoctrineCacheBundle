DoctrineCacheBundle
===================

The DoctrineCacheBundle allows your Symfony application to use different caching
systems through the `Doctrine Cache`_ library.

.. note::

    Since the version 1.3 of DoctrineBundle the DoctrineCacheBundle is included
    by default.
